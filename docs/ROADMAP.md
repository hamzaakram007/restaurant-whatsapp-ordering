# Product Roadmap

## Phase 1 — MVP (implemented)

- WhatsApp menu browsing with numbered categories and items
- Cart, checkout, delivery/takeaway selection
- Manual bank transfer payment with screenshot upload
- Counter dashboard with payment approve/reject and bell alert
- Kitchen display with receipt view and bell alert
- Order status tracking via WhatsApp updates
- Menu admin for basic item management
- Local simulate API for bot testing

## Phase 2 — Operations

- Supabase persistence replacing in-memory store
- Branch support for multi-location restaurants
- Customer profiles and repeat-order shortcuts
- Order analytics and daily sales summary
- Thermal printer / receipt export for kitchen
- Staff authentication for dashboard routes

## Phase 3 — Growth

- Online payment integration (card, wallet, payment link)
- Delivery rider assignment and live rider tracking
- Promo codes, loyalty points, and discounts
- Inventory stock deduction tied to menu availability
- POS / ERP integrations

## Explicitly out of MVP scope

- AI free-form ordering (deterministic state machine only)
- Automatic payment verification
- Multi-tenant SaaS billing
- WhatsApp group ordering
